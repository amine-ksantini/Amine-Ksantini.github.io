##  Fonctionnalités et Design
Ce portfolio a été conçu pour être ultra-rapide, fluide et esthétique, en s'inspirant des codes du design "Apple" (Bento Grid, Glassmorphism, Mode sombre natif).

*  **Single Page Application (SPA) :** Navigation instantanée sans rechargement de page.
*  **Architecture MVC :** Séparation stricte des données (Modèle), de l'affichage (Vue) et de la logique (Contrôleur).
*  **100% Responsive :** Adapté aux écrans d'ordinateur, tablettes et smartphones.
*  **Design Moderne :** Effets de flou (backdrop-filter), animations au survol et micro-interactions.

## 🛠️ Technologies utilisées
* **HTML5** (Sémantique et structure)
* **Tailwind CSS** (Framework utilitaire pour le style)
* **Vanilla JavaScript** (Logique métier et architecture MVC, sans framework lourd)

## 🚀 Lancer le projet en local
Puisqu'il s'agit d'un projet "Vanilla", aucune installation complexe n'est requise :
1. Clonez ce dépôt : `git clone https://github.com/amine-ksantini/amine-ksantini.github.io.git`
2. Ouvrez simplement le fichier `index.html` dans votre navigateur web préféré.
